# NAME

Git::instaweb - run git instaweb on Plack

# SYNOPSIS

    % cd gitdir
    % run-git-instaweb

# INSTALL

    % git clone --recursive git://github.com/shoichikaji/git-instaweb.git
    % cd git-instaweb
    % cpanm .
